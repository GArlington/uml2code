# Whats is UML2Code? #
It is a code to UML generator from JAVA and Python and viceversa (ie. UML to code generator).
## And the features? ##
UML2Code has this features:
<ul>
<li>Extensible</li>
<li>Easy of use</li>
<li>No more time consuming than others!</li>
</ul>
